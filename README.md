# Component Reuse using React.js

## Project Overview

This project demonstrates the concept of Component Reuse in React.js by creating reusable Student Card components. The same component is reused multiple times with different data using props.
The application displays student details such as name, course, and college in a clean card-based UI.

## 🚀 Features

* Reusable React Components
* Passing data using Props
* Multiple student cards using one component
* Clean and responsive UI
* Simple beginner-friendly React project

## 🛠 Technologies Used

* React.js
* JavaScript
* HTML
* CSS
* Git & GitHub

## Installation and Setup

### Step 1: Clone Repository

git clone https://github.com/mummanavasanthi/component-reuse-app.git

### Step 2: Navigate to Project Folder

cd component-reuse-app

### Step 3: Install Dependencies

npm install

### Step 4: Run Application

npm start

## Application Functionality

* Displays multiple student cards
* Reuses the same component for different student data
* Uses props to pass dynamic values
* Demonstrates React component reusability
